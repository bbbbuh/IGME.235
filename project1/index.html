Sword Rush

Sword Rush was an academic project I did in a Game Development class over a semester in a team of 4. The project was done in C# Monogame with the use of GitHub remote repositories as a means of doing collaborative coding. The game is a top-down action rougelike where the objective is to clear as many rooms as possible by defeating all the enemies in each room. The gimick? You are only able to move by lunging forward with an attack. Defeating enemies grants you EXP which helps you level up. When you level up, you pick from 3 randomly assigned attributes to upgrade such as attack damage, attack speed, or health. Defeating enemies and opening chests can also grant you gold. You can use this to purchase powerful equippable powerups in the shop on the main menu. these powerups include a dodge, sheild, and lifesteal.

Contributions

  • Level design and supporting game design
  • External level editor and tiling system
  • Grid positioning system and miscellaneous backend
  • SFX and supporting UI
